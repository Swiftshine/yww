# Yoshi's Woolly World
Yoshi's Woolly World decompilation.

## Setup

1. Obtain Green Hills Software MULTI PowerPC v5.3.27
    - 1. Copy the following folders and directories from GHS into the root directory of this project:
    ```
    cxppc.exe
    ecomppc.exe
    config/license.cfg
    target/ppc.bld
    ```
2. Run `compile.py`
